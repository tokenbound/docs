# createAccount

Deploys the smart contract for a given ERC-6551 account. Uses the Tokenbound wallet implementation by default.

<Tabs items={["viem", "Ethers.js"]}>
  <Tab>
    ```javascript
    import { createAccount } from "@tokenbound/sdk";

    const hash = await createAccount(
      "0xe7134a029cd2fd55f678d6809e64d0b6a0caddcb", // ERC-712 contract address
      "9", // ERC-721 token ID
      walletClient // viem wallet client
    );
    ```

  </Tab>
  <Tab>
    ```javascript
    import { createAccount } from "@tokenbound/sdk-ethers";

    const { hash } = await createAccount(
      "0xe7134a029cd2fd55f678d6809e64d0b6a0caddcb", // ERC-712 contract address
      "9", // ERC-721 token ID
      signer // ethers signer
    );
    ```

  </Tab>
</Tabs>
